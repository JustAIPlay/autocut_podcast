# 视频下载

> 从URL下载视频到本地 /videos 目录

## 快速使用

```
用户: 下载视频 https://xxx.com/video.mp4，脚本号是001
用户: 保存这个视频，脚本号A002
```

## 前置条件

需要提供：
- **视频URL**：视频的完整链接
- **脚本号**：用于命名文件

## 流程

```
1. 确认 /videos 目录存在
    ↓
2. 使用 wget 或 curl 下载视频
    ↓
3. 保存为 /videos/{脚本号}.mp4
    ↓
4. 验证文件完整性
```

## 下载命令

```bash
# 使用 wget
wget -O "videos/{脚本号}.mp4" "{视频URL}"

# 使用 curl
curl -L -o "videos/{脚本号}.mp4" "{视频URL}"

# 使用 yt-dlp（推荐用于平台视频）
yt-dlp -o "videos/{脚本号}.mp4" "{视频URL}"
```

## 支持的视频源

| 来源 | 推荐工具 |
|------|----------|
| 直链MP4 | wget / curl |
| 抖音/快手/B站等 | yt-dlp |
| 飞书云文档 | 先获取下载链接 |

## 输出

```
/videos/001.mp4
/videos/A002.mp4
```

## 常见问题

### Q1: 下载失败
- 检查URL是否可访问
- 检查网络连接
- 尝试使用 yt-dlp

### Q2: 文件不完整
- 检查磁盘空间
- 重新下载

### Q3: 视频格式不支持
- 尝试转换为 MP4 格式
- 使用 FFmpeg 处理
