# 副语言标签快速参考指南

## 📊 使用目标

**40-50% 的对话应该包含标签**（158条对话中约60-80条）

## 🎭 标签类型与使用场景

### 1. `<|laughter|>` 笑声 - 轻松、愉快、认同

**使用场景**：
- ✅ 表示认同："没错 `<|laughter|>` 就是这个道理"
- ✅ 轻松调侃："说得太对了 `<|laughter|>` 大家都懂"
- ✅ 幽默时刻："这可真是 `<|laughter|>` 太有意思了"
- ✅ 互动引导："请点赞 `<|laughter|>` 谢谢大家"
- ✅ 缓解严肃："其实 `<|laughter|>` 也没那么复杂"

**关键词提示**：没错、对、是啊、真的、确实、太好了、太棒了

---

### 2. `<|sigh|>` 叹气 - 感慨、无奈、深思

**使用场景**：
- ✅ 表达感慨："是的 `<|sigh|>` 这个问题很常见"
- ✅ 表示理解："确实 `<|sigh|>` 很多人都这样"
- ✅ 遗憾同情："可惜 `<|sigh|>` 大家都忽略了"
- ✅ 深有体会："唉 `<|sigh|>` 说到心坎里了"
- ✅ 引出话题："哎呀 `<|sigh|>` 这就得说说了"

**关键词提示**：是的、确实、唉、哎呀、可惜、遗憾、真是、居然

---

### 3. `<|breathing|>` 呼吸/停顿 - 思考、转折、列举

**使用场景**：
- ✅ 句子中间停顿："身体就像一条大河 `<|breathing|>` 气血就是活水"
- ✅ 转折前："不过 `<|breathing|>` 我们还要注意一点"
- ✅ 列举项目间："第一是南瓜 `<|breathing|>` 第二是紫米"
- ✅ 强调前："最重要的是 `<|breathing|>` 要坚持"
- ✅ 长句中间："如果你想改善 `<|breathing|>` 就要从根本做起"
- ✅ 思考时："那么 `<|breathing|>` 我们该怎么办呢"

**关键词提示**：那么、然后、接下来、另外、还有、不过、但是、所以

---

## 📍 优先添加位置

1. **感叹词后**：哎呀、唉、哇、嗯
2. **认同词后**：没错、对、是的、确实
3. **转折词后**：但是、不过、然而、可是
4. **列举词后**：那么、然后、接下来、另外
5. **互动引导**：请点赞、欢迎评论、关注我
6. **长句中间**：超过15字的句子

---

## ✅ 使用原则

- ✅ **主动分析每句话的情绪和节奏**
- ✅ **40-50% 的对话要添加标签**
- ✅ 让对话听起来像真人在说话，有呼吸、有情绪
- ❌ 不要在同一句话中添加多个标签
- ❌ 不要机械地添加，要根据语义和情绪

---

## 🎯 情绪分析技巧

### 如何判断应该添加哪个标签？

1. **分析说话人的情绪**
   - 开心、认同 → `<|laughter|>`
   - 感慨、无奈 → `<|sigh|>`
   - 思考、停顿 → `<|breathing|>`

2. **分析句子的节奏**
   - 长句（>15字）→ 中间加 `<|breathing|>`
   - 列举、转折 → 加 `<|breathing|>`
   - 互动、引导 → 加 `<|laughter|>`

3. **分析对话的语气**
   - 轻松愉快 → `<|laughter|>`
   - 严肃深沉 → `<|sigh|>`
   - 平稳叙述 → `<|breathing|>`

## 💡 实战技巧

### 技巧1：长句必加
超过15字的句子，中间一定要加 `<|breathing|>`

**示例**：
```
[S1] 如果你想改善睡眠质量 `<|breathing|>` 就要从调整作息开始。
```

### 技巧2：互动必笑
互动引导（点赞、评论、关注）一定要加 `<|laughter|>`

**示例**：
```
[S2] 请动动您的小手 `<|laughter|>` 给我们点个赞！
```

### 技巧3：感慨必叹
"是的"、"确实"、"唉"等感慨词后一定要加 `<|sigh|>`

**示例**：
```
[S2] 是的 `<|sigh|>` 这个问题确实很普遍。
```

### 技巧4：转折必停
"但是"、"不过"、"然而"等转折词后加 `<|breathing|>`

**示例**：
```
[S1] 方法很简单，不过 `<|breathing|>` 需要坚持。
```

## 📈 质量检查清单

生成播客文案后，检查：

- [ ] 总对话数统计（应该在150-200条）
- [ ] 标签数量统计（应该在60-100个）
- [ ] 标签覆盖率（应该在40-50%）
- [ ] 标签类型分布（笑声、叹气、呼吸都有）
- [ ] 标签位置合理（句首、句中、句尾都有）
- [ ] 情绪表达自然（不机械、不重复）

---

## 🚀 快速上手

**记住这个公式**：

```
情绪标签 = 笑声(认同/互动) + 叹气(感慨/理解)
节奏标签 = 呼吸(停顿/转折/列举)
覆盖率 = 40-50%
```

**一句话总结**：
让播客听起来像真人在说话，有呼吸、有情绪、有节奏！
